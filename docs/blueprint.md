# **App Name**: Scotiabank Event RSVP

## Core Features:

- RSVP Form: Collect attendee information: full name, email, university, career, ID, confirmation. 
- Submission Handling: Handle form submission and send confirmation email.

## Style Guidelines:

- Primary color: Scotiabank Red (#D50032) to reflect brand identity.
- Background color: Off-white (#FAFAFA) for a clean and professional look.
- Accent color: Dark gray (#333333) for text and secondary elements.
- Body and headline font: 'Inter', sans-serif, for a clean and professional look. 
- Prioritize form fields based on importance: Name, Email, University, Career, ID, Confirmation.
- Use a clean and straightforward layout with clear sections and labels.
- Subtle animations or transitions on form submission and confirmation messages.
